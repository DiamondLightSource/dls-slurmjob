Command Line#

dls-slurmjob#

Command line accompanying the dls-slurmjob library.

usage: dls-slurmjob [-h] [--version] [--version-json]

Named Arguments#

--version

Print version string.

--version-json

Print version stack in json.

Default: False

dls_slurmjob_cli.main#

Command line interface to dls-slurmjob.

usage: python -m dls_slurmjob_cli.main [-h] [--log_name string]
                                       {query_jobs,submit_job} ...

Positional Arguments#

subcommand

Possible choices: query_jobs, submit_job

subcommands

Named Arguments#

--log_name

basis name for log

Sub-commands#

query_jobs#

Print list of jobs.

python -m dls_slurmjob_cli.main query_jobs [-h]
                                           [--configuration filename.yaml]
                                           job_id [job_id ...]
Positional Arguments#
job_id

List of jobs.

Named Arguments#
--configuration, -c

Configuration file.

submit_job#

Submit job.

python -m dls_slurmjob_cli.main submit_job [-h]
                                           [--configuration filename.yaml]
                                           filename
Positional Arguments#
filename

Script filename.

Named Arguments#
--configuration, -c

Configuration file.
